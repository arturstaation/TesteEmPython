## Como rodar o projeto
Abra um console e va para a pasta 'src'
```bash
cd src/
```
Instale as bibliotecas necessarias
```bash
pip install -r requirements.txt
```
Defina o caminho do ambiente python
```bash
$env:PYTHONPATH="."
```
Rodar os testes:
```bash
pytest ../tests/
```
